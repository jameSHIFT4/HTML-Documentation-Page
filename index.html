Introduction to HTML

What is HTML?

HTML (HyperText Markup Language) is the standard language used to create web pages. It forms the structure of a webpage by using elements to define content like text, images, and links. HTML is not a programming language but a markup language, which defines the meaning and structure of web content.

HTML Documentation Structure

An HTML document is made up of tags, which enclose different types of content. HTML documents typically begin with a <!DOCTYPE html> declaration, followed by the <html> element, which contains the entire document.

Code Example (HTML Document Structure)

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
     <title>HTML Documentation</title>
   </head>
   <body>
     <h1>Welcome to HTML!</h1>
   </body>
</html>

Key Features of HTM

  • Defines structure of web content
  • Uses tags to enclose elements
  • Not case-sensitive

HTML Tags and Elements

What are HTML Tags?

HTML tags are used to mark up content in a web document. Each HTML tag typically comes in pairs, such as <p> and </p>, where the opening tag begins the element and the closing tag ends it. Some tags, like <img>, are self-closing.

Common HTML Tags

Some of the most commonly used HTML tags include <p> for paragraphs, <h1> through <h6> for headings, and <a> for links.

Code Example (Basic HTML Tags)

<h1>This is a Heading</h1>
<p>This is a paragraph of text.</p>
<a href="https://example.com">This is a link</a>

HTML Attributes

What are HTML Attributes?

HTML attributes provide additional information about elements. They are included in the opening tag and usually come in name/value pairs, such as id="main". Attributes control various aspects of an element, like its size, style, or behavior.

Common Attributes

Some widely used attributes include id, class, href (for links), and src (for images).

Code Example (Using Attributes)

<a href="https://example.com" target="_blank">Visit Example</a>
<img src="image.jpg" alt="A sample image">

List of Common HTML Attributes

  • id: Unique identifier for an element
  • class: Used to group elements for styling
  • href: Specifies the URL of a link
  • src: Specifies the image source

HTML Forms and Input

What are HTML Forms?

HTML forms are used to collect user input. They can contain various input elements like text fields, checkboxes, radio buttons, and submit buttons. The form's data is sent to a server for processing.

Form Elements

Common form elements include <input>, <textarea>, <select>, and <button>.

Code Example (Basic HTML Form)

<form action="/submit" method="POST">
  <label for="name">Name:</label>
  <input type="text" id="name" name="name">
  <input type="submit" value="Submit">
</form>

HTML Links

What are HTML Links?

Links in HTML are created using the <a> (anchor) element. They allow users to navigate from one webpage to another or to a specific section within the same page. Links are defined using the href attribute.

Linking to Other Pages

To link to another webpage, include the URL in the href attribute.

Code Example (HTML Link)

<a href="https://www.example.com">Visit Example</a>

List of Link Types

    margin: auto 0;
  • External Links: <a href="https://example.com">
  • Internal Links: <a href="#section">
  • Email Links: <a href="mailto:someone@example.com">

HTML Semantic Elements

What are Semantic Elements?

Semantic HTML elements clearly describe their meaning in a web document. These elements make webpages more accessible and easier to maintain. Examples include <header>, <footer>, <article>, and <section>.

Common Semantic Elements

Semantic elements help define the structure of a webpage, making it easier for search engines and screen readers to understand the content.

Code Example (Semantic Elements)

<header>
  <h1>Welcome to My Website</h1>
</header>
<article>
  <p>This is an article about HTML.</p>
</article>
<footer>
  <p>© 2024 My Website</p>
</footer>

List of Semantic Elements

  • <header>: Defines a header section
  • <footer>: Defines a footer section
  • <article>: Defines an independent piece of content
  • <section>: Defines a section of content
